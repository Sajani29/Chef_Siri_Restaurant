# 🍽️ Chef Siri Restaurant Website

A responsive and visually appealing restaurant website developed using HTML, CSS, and JavaScript. The website showcases Chef Siri Restaurant's story, menu, gallery, and contact information, providing customers with an engaging online dining experience.

---

## 📌 Features

* 🏠 Home page with welcoming hero section
* 📖 About Us section with restaurant history
* 🍲 Extensive food menu with categories
* 🖼️ Gallery section displaying restaurant images
* 📞 Contact Us section with location and social links
* 🔍 Search bar functionality
* 🛒 Shopping cart icon
* 📱 Responsive navigation menu
* ✨ Lottie animation integration
* 🎨 Modern and attractive user interface

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Boxicons
* Lottie Animation

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/chef-siri-restaurant.git
```

### 2. Open the Project Folder

```bash
cd chef-siri-restaurant
```

### 3. Run the Website

Open `index.html` in your browser.
